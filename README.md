# 💳 TeenPay

TeenPay is a **Flutter-based simulated digital payment application** designed to replicate real-world peer-to-peer transactions among teenagers. It demonstrates core fintech features like authentication, payments, and transaction tracking using Firebase.

---

## 🚀 Features

* 🔐 **Firebase OTP Authentication** for secure login
* 📲 **QR Code-based Payments** for quick transactions
* 📞 **Phone Number-based Transfers** between registered users
* 💸 **Request Money Feature** for peer-to-peer payments
* 🎁 **Rewards System** based on transaction activity
* 📊 **Dashboard with Balance & Transaction History**
* 🪪 **KYC Verification (Simulated)** for user onboarding

---

## 🛠️ Tech Stack

### Mobile App

* Flutter (Dart)

### Backend & Services

* Firebase Authentication
* Cloud Firestore (Real-time database)

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/abhinavNarvekar/TeenPay.git
cd TeenPay
```

### 2. Install dependencies

```bash
flutter pub get
```

### 3. Run the app

```bash
flutter run
```
